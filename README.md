# Tesla Model Y Juniper Delivery Assistant

Mobile-first delivery inspection checklist for a Tesla Model Y Juniper.

## Features

- Interactive delivery-day checklist
- Progress tracking by section
- Critical issue logging
- Notes for checklist items and sections
- Photo placeholders
- Printable delivery report
- Light and dark mode
- Local progress saving on your device
- Offline support after the app is loaded from an HTTPS host

## How To Use

Open the app on your phone, complete each inspection item during delivery, add notes for anything important, and mark any serious problem as failed. Failed items automatically appear in the Critical Issues page.

For the best phone experience, publish this repository with GitHub Pages, open the GitHub Pages URL on your phone, then add it to your home screen.

## Privacy

Inspection progress, notes, and VIN are saved only in your browser's local storage. They are not sent anywhere by this app.
